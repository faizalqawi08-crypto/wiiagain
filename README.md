# ET Store - Portal Cuti Karyawan

Sistem pengajuan cuti karyawan ET Store berbasis PHP sederhana dengan fitur:

- Pengajuan cuti
- Upload lampiran surat
- Status cuti untuk user
- Dashboard admin untuk verifikasi
- Login admin dan user

## Struktur Folder

- admin/ : halaman admin
- user/ : halaman user
- uploads/ : tempat penyimpanan file lampiran
- assets/ : aset visual seperti logo

## Cara Menjalankan

1. Jalankan server Apache melalui Laragon
2. Arahkan ke http://localhost/cuti/
3. Login admin melalui /admin/login.php
4. Login user melalui /user/login.php

## Catatan

Pastikan database sudah dibuat sesuai file database.sql.
